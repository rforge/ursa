rmapRasterR Documentation

Definition of rmapRaster class.

Description

rmapRaster is S3 class for manipulation with georeferred raster images. See ‘Value’ section.
is.rmap checks inhering to class rmapRaster

Usage

## S3 method for class 'rmapRaster'
print(x, digits = NA, grid = FALSE, raw = FALSE, ...)

## S3 method for class 'rmapRaster'
str(object,...)

is.rmap(obj, ref = NULL)

is_rmap(obj, ref = NULL)

Arguments

x, object

rmapRaster object.

obj

Any.

digits

Passed to format function

grid

Logical. If grid=TRUE then returns simplified metadata.

raw

Logical. If FALSE and values are categories, then attempting to restore numeric values from categorical names is before calculating of statistics. If TRUE then values for statistics are used as is. Default is FALSE.

...

Passed to format function

ref

Character or NULL. If character, then checking of rmapRaster sub-class(es):

Pattern Description
NULL rmapRaster
(raster|brick|rmap) rmapRaster
grid rmapGrid
(ct|color|table) rmapColorTable
stack rmapStack
con rmapConnection
val rmapNumeric OR rmapCategory
cat rmapCategory

Details

is.rmap() is designed mainly for developers to check arguments' class in function's call. is_rmap is a synonym to is.rmap.
Stucture of rmapRaster class is generated by non-public .raster.skeleton() function.

Value

rmapRaster is R's S3 class. It is a list with items:

grid

Geospatial properties. rmapGrid object

con

Connection properties.rmapConnection object

value

2-dimensional numerical or integer matrix of classes rmapValue in (spatial, temporal) specification formed from (samples*lines, bands). If data are not in memory, then NA.

dim

Dimension of $value. If bands are interpeted as observations in time, then it is spatial by temporal dimension of data. Even data are not in memory, dim is a dimension of whole data.

name

Band names

colortable

Color table. rmapColorTable object

is.rmap(x) returns TRUE, if class of x is rmapRaster
