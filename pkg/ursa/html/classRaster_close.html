closeR Documentation

Close connections for files with data

Description

close() for rmapRaster object closes connection for opened file using inheritted function base::close. Function envi.close() closes opened connection for ENVI binary file.

Usage

## S3 method for class 'rmapRaster'
close(...)

envi.close(...)

Arguments

...

Object or sequence of objects of class rmapRaster.

Value

NULL

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

close of base package

Examples

session.grid(NULL)
a <- envi.create()
fname <- a$con$fname
message(paste("Created file",dQuote(basename(fname)),"will be deleted."))
print(dir(pattern=basename(envi.list(fname))))
close(a)
invisible(envi.remove(fname))
