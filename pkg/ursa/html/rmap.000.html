rmapR Documentation

Get and set properties of raster image.

Description

rmap is a wrapper to initialize object of class rmapRaster, to get and to set properties for this object.

Usage

rmap(obj, attr, ...)
rmap(obj, attr, ...) <- value

Arguments

obj

Any numeric structure (scalar, matrix, array) for initializing. Object of class rmapRaster for extracting and assigning properties.

attr

Character. Name of property.

...

Arguments, which are passed for properties specification.

value

Value for property assignment.

Details

Initializing function rmap with missing argument attr is a wrapper for function as.rmap.

Matched pattern Replace method? Description of property Implementation
"grid" Yes Raster grid (extent, projection, cellsize) rmap.grid
"(proj|crs)" Yes Coordinate reference system rmap.proj
"val" Yes Raster value in the internal storage format rmap.value
"(colort|ct)" Yes Color table rmap.colortable
"(categ|class)" Yes Names of categories names(rmap.colortable(obj))
"name" Yes Band names names
"^table$" No Frequency of unique values as.table
"cell" No Squared cell size with(rmap.grid(obj),sqrt(resx*resy))
"^dim$" No Dimension of raster image dim
"(extent|bbox)" No Spatial extent of raster image with(rmap.grid(obj),c(xmin=minx,ymin=miny,xmax=maxx,ymax=maxy))
"(nrow|rows|lines)" No Number of rows of raster image rmap.grid(obj)$rows
"(ncol|columns|samples)" No Number of columns of raster image rmap.grid(obj)$columns
"con" No structure of connection obj$con

Argument ... is used to specify band index or band pattern in rmap(obj,"value",...)

Value

Inititalizing function rmap (missing attr) returns object of class rmapRaster.

Extract function rmap returns object of respective property.

Replace function rmap<- returns object

Examples

a1 <- rmap(volcano)
print(a1)
display(a1)

a2 <- rmap(volcano,flip=TRUE)
print(a2)
display(a2)

a3 <- rmap(volcano,permute=TRUE)
print(a3)
display(a3)

a4 <- rmap(volcano,flip=TRUE,permute=TRUE)
print(a4)
display(a4)

dima <- c(200,300,4)
b1 <- rmap(array(runif(prod(dima)),dim=dima))
print(b1)
display.brick(b1,scale=1,pal.r=0,pal.hue=0,decor=FALSE)

session.grid(NULL)

c1 <- rmap(seq(3))
print(c1)
c2 <- rmap(seq(3),bands=3)
print(c2)

c3 <- rmap(value=FALSE)
str(rmap(c3,"value"))

c4 <- rmap(bands=2,nodata=-99L)
print(c4)
print(rmap(c4,"nodata"))

c5 <- rmap(bandname=format(Sys.Date()+seq(7)-1,"%A"))
rmap(c5,"value") <- rev(seq(nband(c5)))
c5 <- colorize(c5)
ct <- rmap(c5,"colortable")
print(c5)

v <- rmap(c5[3:5],"value")
str(v)
v <- c(v)
str(v)
c6 <- rmap(v,colortable=ct)
print(c6)
print(rmap(c6,"colortable"))
