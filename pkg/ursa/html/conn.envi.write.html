envi.writeR Documentation

Write raster image to ENVI .hdr Labelled Raster file.

Description

envi.write writes in-memory object of class rmapRaster to disk in the ENVI .hdr Labelled Raster file format.

Usage

envi.write(obj, ...)

Arguments

obj

Object of class rmapRaster.

...

Arguments, which are passed to envi.create. Usually, only file name (character) is required. If missing, then occasional name is assigned.

Details

envi.write implements writting the whole rmapRaster object to disk. For multiple access to disk (by chunks), use followed construction: 

a <- envi.create(fname)
a[condition_1] <- value1
a[condition_2] <- value2
...
close(a)

Value

Integer code of ENVI data type. See values of the “data type” field in description of the ENVI Header Format.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

envi.create, Replace method [<- for rmapRaster object, envi.close (close for rmapRaster object).

gdal.write(...,driver="ENVI") uses GDAL (rgdal) for writting rmapRaster object to the ENVI .hdr Labelled Raster file.

Examples

session.grid(NULL)
list1a <- envi.list(tempdir())
list1b <- envi.list()
fname <- tempfile()
a <- rmap.dummy()
bandname(a) <- c("first","second","third")
envi.write(a)
envi.write(a,fname)
list2a <- envi.list(tempdir())
list2b <- envi.list()
fname1 <- list2a[!(list2a %in% list1a)]
fname2 <- list2b[!(list2b %in% list1b)]
a2 <- envi.open(fname1)
print(a2)
close(a2)
envi.remove(c(fname1,fname2))
