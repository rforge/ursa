rmapGridR Documentation

Spatial paramaters of raster images.

Description

Class rmapGrid is a part of class rmapRaster. It defines spatial locations of image.

Usage

## S3 method for class 'rmapGrid'
print(x, ...)

## S3 method for class 'rmapGrid'
str(object, ...)

## S3 method for class 'rmapGrid'
dim(x)

Arguments

x

rmapGrid object in function print.

object

rmapGrid object in function str.

...

Further arguments passed to generic functions print and str.

Details

The blank rmapGrid object is generated by calling of rmap.grid() without arguments. These approaches provide unified sequence of list's items:

 $ maxy   : num NA

 - attr(*, "class")= chr "rmapGrid"
NULL

Value

Object of class rmapGrid is a list with items:

columns

Number of columns (samples)

rows

Number of rows (lines)

resx

Grid cell size by horizontal axis

resy

Grid cell size by vertical axis

minx

Left margin of boundary box

maxx

Right margin of boundary box

miny

Bottom margin of boundary box

maxy

Top margin of boundary box

proj4

PROJ.4 string

Function dim for object of class rmapGrid returns named vector of length 2: number of rows ("lines") and number of elements in a row ("samples")
