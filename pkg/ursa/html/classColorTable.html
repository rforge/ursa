colortableR Documentation

Color Tables of raster images.

Description

Manipulation with color tables of raster images.

Usage

## S3 method for class 'rmapColorTable'
print(x, ...)

## S3 method for class 'rmapColorTable'
x[i]


rmap.colortable(x)

rmap.colortable(x) <- value

Arguments

x

rmapRaster object.

value

Named character vector. Values are colors in “#RRGGBB” notation or R color names (colors). names(value) are names of categories.

i

Integer vector. Indices specifying elements to extract part (subset) of color table.

...

Passing to generic print. Currently not used.

Details

The example of the class structure

Class 'rmapColorTable'  Named chr [1:4] "#313695" "#BCE1EE" "#FDBE70" "#A50026"
  ..- attr(*, "names")= chr [1:4] "<= 450" "(450;550]" "(550;650]" "> 650"

It is recommended to use rmap.colortable and rmap.colortable<- instead of colortable and colortable<-. rmap.colortable and colortable are synonyms. rmap.colortable<- and colortable<- are synonyms too. Package raster contains colortable and colortable<- functions. colortable and colortable<- will be remove from this package if the case of frequent joint use of both packages.

If color tables describe continuous and non-intersecting intervals, then print gives additional line of extracted breaks.

Value

rmap.colortable returns value of $colortable element if rmapRaster object.

rmap.colortable<- returns rmapRaster object with modified $colortable element.

Class of $colortable element is “rmapColorTable”. This is named character vector, where names are categories, and values are “#RRGGBB” or R color names.

Extract function [] for rmapColorTable object returns object of class rmapColorTable.

Color tables are written to ENVI header file.

Warning

If colors are specified as R color names, then slow down may appear.

Examples

session.grid(NULL)
## Not run: print(methods(class="rmapColorTable"))

a <- pixelsize()
print(a)
b1 <- colorize(a,value=c(400,500,600,700),interval=FALSE)
b2 <- colorize(a,value=c(450,550,650)    ,interval=TRUE)
## Not run: display(list(b1,b2))
print(is.rmap(a,"colortable"))
print(is.rmap(b1,"colortable"))
print(is.rmap(b2,"colortable"))
print(rmap.colortable(a))
print(rmap.colortable(b1))
print(rmap.colortable(b2))
rmap.colortable(b2) <- c("Low"="darkolivegreen1"
                        ,"Moderate"="darkolivegreen2"
                        ,"High"="darkolivegreen3"
                        ,"errata"="darkolivegreen4")
print(rmap.colortable(b2))
names(rmap.colortable(b2))[4] <- "Polar"
print(rmap.colortable(b2))
## Not run: display(b2)
