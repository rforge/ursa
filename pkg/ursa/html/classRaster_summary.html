summaryR Documentation

Summary of raster image.

Description

Function summary for rmapRaster object produces summaries for each band.
Function summary for rmapValue object produces summaries for all values of raster image regardless of bands.

Usage

## S3 method for class 'rmapRaster'
summary(x, ...)

## S3 method for class 'rmapNumeric'
summary(x, ...)

## S3 method for class 'rmapCategory'
summary(x, ...)

Arguments

x

Object of classes rmapRaster, rmapNumeric, or rmapCategory

...

Additional arguments affecting the summary produced.

Details

summary for rmapRaster object applies summary to each column of two-dimensions value matrix and collating the results.
summary for rmapValue object drops dimensions and applies summary to a vector.

Value

summary for rmapRaster object returns value of function summary.matrix.
summary for rmapValue object returns object of class \dQuote{summaryDefault}.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

summary in package base.

Examples

session.grid(NULL)
session.grid(regrid(mul=1/4))
a <- rmap.dummy(nband=3)
print(summary(a))
print(summary(rmap.value(a)))
print(a)
