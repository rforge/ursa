rmap.projR Documentation

Extract and assign projection of raster images.

Description

Functions manipulate with $proj4 item of the rmapGrid object, which is embedded in the rmapRaster object (obj$grid$proj4). Projection is specified in PROJ.4 notation.

Usage

rmap.proj(obj)
rmap.proj(obj, keepGrid = FALSE) <- value

Arguments

obj

rmapRaster object. It is allowed rmapGrid object for rmap.proj (Extract) function.

keepGrid

Logical. Should sessional grid be changed after assignment. Default is FALSE.

value

Character sting in PROJ.4 format.

Value

Extract function rmap.proj returns character value of $grid$proj4 item of rmapRaster object.

Replace function rmap.proj<- returns rmapRaster with modified $grid$proj4 item.

Author(s)

Nikita Platonov platonov@sevin.ru

Examples

session.grid(NULL)
a <- rmap.dummy(nband=1)
print(rmap.proj(a))
p4s <- "+init=epsg:3576"
rmap.proj(a) <- p4s
print(rmap.proj(a))
fname <- tempfile()
envi.write(a,fname)
a2 <- envi.read(fname,resetGrid=TRUE)
print(rmap.proj(a2))
print(rgdal::CRSargs(sp::CRS(p4s)))
envi.remove(fname)
