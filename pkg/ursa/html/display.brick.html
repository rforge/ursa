display.brickR Documentation

Plot multi-band homogenous raster image in the PNG format.

Description

Raster image is forced to be interpeted as homogenuous (having the same units). It implies creating multi-panel layout with multiple colorbars.

Usage

display.brick(obj, ...)

Arguments

obj

Object of class rmapRaster or list of rmapRaster objects.

...

Passed to hierarchy of plotting functions:

  • compose.open

    • compose.design

  • compose.plot

    • panel.new

    • panel.raster

    • panel.decor

      • panel.gridline

      • panel.coastline

      • panel.scalebar

      • panel.annotation

    • compose.legend

  • compose.close

Details

If argument obj is list of rmapRaster objects (or object of class rmapStack) then obj is coerced to class rmapRaster ('stack' is coerced to 'brick').

Value

Function returns NULL value.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

display, display.stack, display.rgb

Examples

session.grid(NULL)
a <- rmap.dummy(nband=3,min=0,max=250)
a[2] <- -a[1]
a[3] <- sqrt(a[1])
## Not run:  a <- rmap.stack(a) # all the same, further coercion to 'brick'
print(a)
## Not run: display(a) # likely 'display.stack' will be called 
display.brick(a,stretch="eq",labels=c(-150,-100,0,10,12,20,100,150))
