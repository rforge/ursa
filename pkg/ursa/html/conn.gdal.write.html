gdal.writeR Documentation

Write raster image to GDAL file(s)

Description

gdal.write writes in-memory object of class rmapRaster to disk using GDAL from rgdal package.

Usage

gdal.write(obj, ...)
rmap.write(obj, fname)

Arguments

obj

Object of class rmapRaster.

...

Arguments, which are passed to gdal.create. Usually, only file name with extension (character) is required. If extension is ".envi", then GDAL driver "ENVI" is used. If extension is ".tif", then GDAL driver "GTiff" is used. If extension is ".img", then GDAL driver "HFA" is used. If extension is ".jpg" or "*.jpeg", then GDAL driver "JPEG" is used. If extension is ".bmp", then GDAL driver "BMP" is used. If extension is ".png", then GDAL driver "PNG" is used. Additionally, argument driver should be specified. If argument ... is missing, then occasional name is assigned.

fname

Character. File name with extension.

Details

rmap.write is simplified call of gdal.write.

gdal.write implements writting the whole rmapRaster object to disk. For multiple access to disk (by chunks), use followed Replace construction: 

a <- gdal.create(fname)
a[condition_1] <- value1
a[condition_2] <- value2
...
close(a)

Value

Integer code of ENVI data type. See values of the “data type” field in description of the ENVI Header Format.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

gdal.create, Replace method [<- for rmapRaster object, close method for rmapRaster object.

envi.write

Examples

session.grid(NULL)
ftemp <- tempfile(pattern="",fileext="")
fpath <- dirname(ftemp)
fname <- basename(ftemp)
a <- round(rmap.dummy(1,min=0,max=255,nodata=NA))
envi.write(a,file.path(fpath,paste0(fname,"_1",".envi")))
gdal.write(a,file.path(fpath,paste0(fname,"_2")))
gdal.write(a,file.path(fpath,paste0(fname,"_3",".tif")))
gdal.write(a,file.path(fpath,paste0(fname,"_4")),driver="EHdr")
flist <- dir(path=fpath,pattern=fname,full.names=TRUE)
file.remove(flist)
blist <- basename(flist)
res <- NULL
for (i in seq(4))
   res <- c(res,paste(grep(paste0("_",i),blist,value=TRUE),collapse=" "))
print(res)
