dimR Documentation

Dimension of multiband raster image

Description

Retrieve the dimension of an object of class rmapRaster. The replacement function is dummy; it doesn't change raster dimension.

Usage

## S3 method for class 'rmapRaster'
dim(x)
## S3 replacement method for class 'rmapRaster'
dim(x) <- value

Arguments

x

Object of class rmapRaster

value

Any. Ignored

Details

Use extract operator [] and combine function c to change third (e.g., temporal) dimension of raster.
Use regrid function to change grid parameters and to resize/resample raster into new grid.

Value

The 'Extract' function dim returns named integer vector of length three: 1) number of lines/rows, 2) number of samples/columns, 3) number of bands/channels/layers.
The 'Replacement' function dim<- returns rmapRaster object without changes.

Author(s)

Nikita Platonov platonov@sevin.ru

Examples

session.grid(NULL)
session.grid(regrid(mul=1/16))
a <- rmap.dummy(nband=3)
rmap.info(a)
print(dim(a))
dim(a) <- c(25,00,34)
print(dim(a))
b <- envi.create("tmp1",bandname=letters[1:5],compress=FALSE)
print(dim(b))
close(b)
envi.remove("tmp1")
