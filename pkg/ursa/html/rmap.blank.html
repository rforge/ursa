blankR Documentation

Does any band contain no information?

Description

Set of functions for checking is any or all bands have no data, and for retrieving indices for non-data bands.

Usage

band.blank(obj)

rmap.blank(obj)

Arguments

obj

Object of class rmapRaster

Details

It is defined locally that if all values of band are NA or 0, then such band band.blank. The fact is rmap.new create new object in memory with default values NA, but envi.create writes zeros to disk quick. It is decided to consider both these cases as blank. Function band.blank checks blanks for each band of image. If all bands are blank then function rmap.blank returns TRUE.

Value

Function rmap.blank returns logical value of length 1.

Function band.blank returns logical value of length nband(obj).

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

is.na returns object of class rmapRaster; it is mask of cells, which have NA value.

Examples

session.grid(NULL)
a <- rmap.new(bandname=c("first","second","third","fourth"))
rmap.value(a,"first") <- 0 ## 'a[1] <- 1' works, but it is slow
print(rmap.blank(a))
a[3] <- pixelsize()
a[4] <- a[3]>625
print(a)
print(band.blank(a))
print(which(band.blank(a)))
print(rmap.blank(a))
