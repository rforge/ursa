rmapProgressBarR Documentation

Progress bar

Description

Informative progress bars with dispaying elapsed and remained time.

Usage

rmapProgressBar(kind = c("tk", "win", "txt"),
                title = basename(strsplit(commandArgs(FALSE)[4],"=")[[1]][2]),
                label = "", min = 0, max = 1, initial = min, width = NA, style = 1)

setRmapProgressBar(pb, value, title = NULL, label = NULL)

Arguments

kind

Character. Type or progress bar.

title
label
min
max
initial
width
style
pb
value

Details

Wrapper to one of txtProgressBar, winProgressBar, tkProgressBar.

Method close(con,...) depends on class of progress bar.

Visualization of progress bar is updates each 0.5 seconds, it is efeective for multiple short-term iterations.

Value

rmapProgressBar returns object of refererence progress bar.

Note

Function name in style camelCase for consistence with other progress bar functions in R.

Examples

session.grid(NULL)
n1 <- 3
n2 <- 683
p <- 0.0051
a <- ls.str()
require(tcltk)
pb <- rmapProgressBar(min=0,max=n1,title="first")
for (i in seq(n1)) {
   setRmapProgressBar(pb)
   pb2 <- plProgressBar(min=0,max=n2,title="second")
   for (i in seq(n2)) {
      setRmapProgressBar(pb2)
      Sys.sleep(p)
   }
   close(pb2)
}
close(pb)
