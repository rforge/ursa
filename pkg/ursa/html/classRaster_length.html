nbandR Documentation

Get number of bands of raster image.

Description

nband (length) returns number of bands (layers, if appropriate in terminology) of rmapRaster object.

Usage

nband(obj)

## S3 method for class 'rmapRaster'
length(x)

Arguments

x

Object of class rmapRaster

Details

length for rmapRaster object is a synonym for nband.

Value

Positive integer of length 1.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

bandname (names for rmapRaster object).

Examples

session.grid(NULL)
a1 <- pixelsize()
print(a1)
print(nband(a1))
a2 <- c("Band 1"=a1,Band2=a1/2,sqrt=sqrt(a1),NA)
print(a2)
print(nband(a2))
