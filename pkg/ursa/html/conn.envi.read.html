envi.readR Documentation

Read ENVI .hdr Labelled Raster file to memory

Description

Reads all or several bands of ENVI .hdr Labelled Raster file from disk to memory.

Usage

envi.read(fname, ...)

Arguments

fname

Character. Filename for ENVI .hdr Labelled Raster file.

...

For readENVI: arguments, which are passed to envi.read function.

For envi.read: Set of argumetns, which are recognized via their names (using regular expressions) and classes. In the case of grids mismatch some arguments (e.g., resample) are passed to regrid function.

(subset)*

Name can be omitted. Integer or character. If integer, then indices of bands, either positive or negative. Positive indices are for included bands, negative indices for omitted bands. If character, then either sequence of band names or regex string. By default (subset=NULL), function reads all bands.

(ref)*

Name can be omitted. rmapRaster or rmapGrid object. Reference grid for raster image resizing. By default (ref=NULL) there is no resizing.

(nodata|ignorevalue)

Numeric. Value, which is ignored. By default (nodata=NaN) igrore value is taken from ENVI metadata (*.hdr or *.aux.xml).

reset(Grid)*

Logical. If TRUE, then session grid is ignored, and new session grid is assigned from input file. If FALSE, then input file is nested in the session grid.

verb(ose)*

Logical. Value TRUE may provide some additional information on console. Default is FALSE.

Details

Function envi.read is designed to one-time reading (from disk to memory) ENVI .hdr Labelled Raster file. For multiple access to disk (by chunks), use followed construction: 

a <- envi.open(fname)
d1 <- a[condition_1]
d2 <- a[condition_2]
...
close(a)

In this case, the connection keeps open. The gain is more effective for compressed binary files.

Value

Object of class rmapRaster.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

envi.open, Extract method [ for rmapRaster object, envi.close.

gdal.read uses GDAL (rgdal) to read ENVI .hdr Labelled Raster file.

Examples

session.grid(NULL)
fname <- tempfile()
a <- rmap.dummy()
bandname(a) <- c("first","second","third")
envi.write(a,fname,compress=FALSE)

print(envi.read(fname))
print(envi.read(fname,c(1,3)))
print(envi.read(fname,-c(1,3)))
print(envi.read(fname,c("first","third")))
print(envi.read(fname,"iR"))

print(session.grid())
g <- regrid(session.grid(),mul=1/2.3)
b <- envi.read(fname,ref=g)
print(session.grid())
print(b)

envi.remove(fname)
